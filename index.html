Ticker Board

This will be the landing page for my ticker board plugin.

Features include:

  • Vanilla JS
  • CSS included
  • Update message methods
  • That's all so far

It's currently not accessible. There's no way a screen reader would have a good time reading these, for example. It is compatible with copying to clipboard, however. Hello.

Known bugs include:

  • A race condition, whereby the result of firing off a change to a ticker board row while it's already processing one results in some letters potentially reflecting the earlier message
